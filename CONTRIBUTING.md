# Contributing to Zen-Captcha-Core

First off, thanks for taking the time to contribute! We want to keep this project "Neon Stable" and accessible to everyone.

## 🚀 How Can I Contribute?

### Reporting Bugs
* Check the **Issues** tab to see if the bug has already been reported.
* If not, open a new issue using the provided template. 
* Include your browser version and a screenshot if the "Neon Glow" is looking glitchy.

### Suggesting Enhancements
* We love new ideas! Whether it's a new theme (Matrix Green?) or a performance tweak, open an issue first to discuss the "vibe" before writing the code.

### Pull Requests
1. **Fork the repo OR clone** and create your branch from `main`.
2. **Stick to the Style**: Keep the glassmorphism and CSS variables consistent. 
3. **Document**: If you add a new feature, update the `README.md` or the `Chanelog`.
4. **Issue First**: Ensure there is an open issue for the work you are doing so we don't double up on efforts.

## 🎨 Design Guidelines
* **Color Palette**: Stick to high-contrast neon shades.
* **Glassmorphism**: Maintain the `backdrop-filter: blur()` standards currently in the CSS.
* **Simplicity**: This is a frontend-only gateway. Avoid adding heavy external dependencies.

## 📜 Code of Conduct
By participating in this project, you agree to abide by our [Code of Conduct](github.com/afifashamsadvivo/CODE_OF_CONDUCT.md).
