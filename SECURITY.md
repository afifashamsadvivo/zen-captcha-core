# Security Policy

## Supported Versions

We currently provide security updates for the following versions of Zen-Captcha-Core:

| Version | Supported          |
| ------- | ------------------ |
| > 1.0   | ✅ Yes             |
| < 1.0   | ❌ No              |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security-related bug or a way to bypass the captcha logic, please help us keep the project secure by reporting it privately. 

You can report vulnerabilities by:
1. Opening a [**Draft Security Advisory**](https://github.com/afifashamsadvivo/Zen-Captcha-Core/security/advisories/new) on GitHub (under the "Security" tab).
2. Sending an email to **[here](mailto:afifashamsad.vivo@gmail.com?subject=[SECURITY]%20Vulnerability%20Report%20-%20Zen-Captcha-Core&body=Description%3A%0A%0ASteps%20to%20Reproduce%3A%0A1.%0A2.%0A3.%0A%0AAffected%20Component%20(CSS/JS/Logic)%3A%0A%0APotential%20Impact%3A%0A%0AEnvironment%20(Browser/OS)%3A)**.

We will acknowledge your report within 48 hours and provide a timeline for a fix. Thank you for helping keep the neon grid safe!
